# 🎵 The Band Website Clone - W3Schools (Practice Project)

This is a **practice project** that replicates the "The Band" website from **W3Schools**. The goal of this project is to reinforce my understanding of **HTML, CSS, and basic JavaScript** through hands-on coding — not for commercial or portfolio purposes.

---

## 📌 Project Goals

- ✅ Practice real-world layout building using HTML & CSS
- ✅ Explore Flexbox and responsive techniques
- ✅ Recreate common UI components such as modals, buttons, navigation, and grid layouts
- ❌ Not intended for commercial use
- ❌ Not part of any freelance or job interview portfolio

---

## 🔧 Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla)
- [Themify Icons](https://themify.me/themify-icons) for UI icons

---
